Техническая задача:
"Разработать веб-приложение на языке PHP, позволяющее создавать, просматривать, модифицировать и удалять записи. Каждая запись должна содержать заголовок, имя автора, произвольный текст, дату создания и опционально изображение (файл). Записи должны храниться в базе данных."

Задача решена с использованием Yii 2 basic.

Для тестирования приложения нужно настроить подключение к базе данных в config/db.php, также нужно добавить в БД таблицу records.sql находящуюся в корневой директории.
Чтобы запустить приложение, запустите basic/web/index.php

Для создания базовой структуры была использована Gii, особо с видами и мелочами не заморачивался, "Суть теста - продемонстрировать понимание ООП и PHP:".
Из-за того, что давно не занимался Yii 2, заняло время на вспоминание и чтение документации.

Контроллер: "controllers/RecordsController.php"
Модели: models/
    ImagePath - вычисления пути к фото по id записи
    Records - ActiveRecord таблицы records
    RecordsSearch - модель для поиска записей по столбцам (создана Gii)
    UploadForm - модель, сохраняющая фото

Все виды находятся в views/Records

Фото сохраняются в web/img, название фото равно id записи.
